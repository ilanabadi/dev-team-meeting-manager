# MeetMe
## Dev Teams Meetings Manager
**A school project for JOHN BRYCE. MeetMe is an app that manages development teams meetings.**

## Key Features
* Choose a Dev team and see its corrent meetings
* Schedule new meetings for any of the dev teams
* On the chosen dev team meeting display you can see the duration of that meeting

## Tech Stack
* Backend: node.js, MySQL
* Frontend: React, Bootstrap
* Logic: Typescript

## How to install the project
This project uses a MySQL database. In order to fully experience it, do this:
1. After cloning the project make sure your Docker is running
2. Open your terminal in the project folder and run:
```bash
docker compose up -d
```
3. Open new terminal in the project backend folder and install package.json
```bash
npm i
```
4. Open new terminal in the project frontend folder and install package.json
```bash
npm i
```
5. In the terminal where backend is open run this code:
```bash
npm start
```
6. Do the same In the terminal where frontend is open:
```bash
npm start
```
7. The app will now open on your browser and and you may test it for yourself.
